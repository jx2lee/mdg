맥(Mac) 또는 macOS에서는 MBR(Master Boot Record) 대신 주로 GPT(GUID Partition Table)를 사용합니다. GPT는 MBR의 현대적인 대안으로, 더 큰 디스크와 더 많은 파티션을 지원합니다.

macOS에서 사용되는 관련 용어로는:

1. Apple Partition Map (APM): 옛 PowerPC 기반 Mac에서 사용되던 파티션 구조입니다.
2. EFI System Partition (ESP): GPT 시스템에서 부팅에 필요한 파일을 저장하는 특별한 파티션입니다.
3. CoreStorage: 논리적 볼륨 관리를 위한 Apple의 기술입니다.
4. APFS (Apple File System): Apple이 개발한 최신 파일 시스템으로, HFS+를 대체했습니다.

이러한 용어들이 Mac 시스템에서 MBR과 관련된 개념을 대체하거나 보완하는 역할을 합니다. 더 자세한 정보가 필요하시다면 말씀해 주세요.
